Scrlr - Help

(2009-05-09 現在、まだ書き中)

Scrlr (スクローラー) は、ウェブを自動でスクロールして眺めるためのページです。

Memorium などのウェブを眺めるためのソフトウェアが好きなので、真似して作ってみました。

関連するものに gasher があります。

動作

  • 基本的には全自動です。検索キーワードが自動で決められ、検索され、結果が表示されます。
  • 画面の左半分では、ウェブ検索した結果が、下から上へスクロールします。
  • 画面の右半分では、イメージ検索して取得した結果が、画像とともに下から上へスクロールします。
  • 画面の上部にヘッダーバーがあります。このバーは使われないと自動で隠れます。再びヘッダーバーを表示させるには、マウスカーソルを近づけます。
  • ヘッダーバーには直近の検索で使われたキーワードや、 Scrlr の状態が表示されます。
  • ヘッダーバーをクリックすると、一時停止状態 [||] になります。 再びクリックするとスクロールが再開 [>>] します。
  • 検索結果のテキストをクリックすると、そのテキストのページが新しいウィンドウで開きます。 そのページを見つけたキーワードの、検索結果のページも開きます。 そしてスクロールは一時停止状態になります。
  • 検索キーワードは次のように決まります。最初は三日前の日付で検索します。 その検索結果のテキスト中からキーワードがランダムに選ばれます。 以降は、「キーワードで検索」、「検索結果のテキストからランダムにキーワードを拾う」、 「それでまた検索する」の繰り返しになります。
  • 検索結果が 0 件になった場合は、数十秒待った後に異なるキーワードで検索を試みます。 待っている間は表示するデータがないので、スクロールは止まります。

おすすめの使い方

Scrlr のおすすめの使い方は、ブラウザを全画面表示にしてスクリーンセーバの代わりにすることです。 デュアル・ディスプレイ環境で、使ってない片面に表示させるなどもいいかもしれません。

IE8 や Opera ブラウザでは、ファンクションキーの F11 などで、全画面表示に切り替えられます。

Firefox では、全画面表示にしてもアドレスバーやタブバーが残ります。ページの内容のみの全画面表示にするには Full Fullscreen :: Firefox Add-ons などのアドオンを使う必要があります。
