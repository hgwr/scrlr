Scrlr - Help

(2009-05-09 現在、まだ書き中)

Scrlr (スクローラー) は、ニュースやウェブを自動でスクロールして眺めるためのソフトウェアです。

Memorium などのウェブを眺めるためのソフトウェアが好きなので、真似して作ってみました。

関連するものに gasher があります。

動作

  • 基本的には全自動です。
  • 画面の左半分では、Scrlr がウェブ検索した結果が、下から上へスクロールします。
  • 画面の右半分は、イメージ検索して取得した結果が画像とともに、下から上へスクロールします。
  • 画面の上部にヘッダーバーがあります。このバーは使われないと自動で隠れます。マウスカーソルを画面の上部に持っていくと、またヘッダーバーが現れます。このヘッダーバーには直近の検索で使われたキーワードなどが表示されます。
  • 画面をクリックすると、一時停止状態とスクロールする状態が切り替わります。
  • 検索結果のテキストをクリックすると、そのテキストのページが新しいウィンドウで開きます。
  • 検索キーワードは次のように決まります。一番最初に三日前のニュースが検索され、そのテキスト中からキーワードがランダムに選ばれます。以降は、「キーワードで検索」、「検索結果のテキストからランダムにキーワードを拾う」、「それでまた検索する」の繰り返しになります。
  • ランダムに選んだキーワードで検索結果の数が 0 件になった場合は、数十秒待ったあとに違うキーワードで検索を試みます。待っている間は表示するデータがないので、スクロールは止まります。

おすすめの使い方

Scrlr のおすすめの使い方は、ブラウザを全画面表示してスクリーンセーバの代わりにすることです。 デュアル・ディスプレイ環境で、使ってない片面に表示させるなどもいいかもしれません。

IE8 や Opera ブラウザでは、ファンクションキーの F11 などで、全画面表示に切り替えられます。

Firefox では、全画面表示にしてもアドレスバーやタブバーが残ります。ページの内容のみの全画面表示にするには Full Fullscreen :: Firefox Add-ons などのアドオンを使う必要があります。
